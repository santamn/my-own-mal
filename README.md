# My Own Mal

[mal (Make-A-Lisp)](https://github.com/kanaka/mal)の自分用の実装です。

## Rust

### Build

impls/rustディレクトリで

```
$ make stepX
```

### Run the REPL

トップレベルで
```
$ impls/rust/target/release/stepA_mal
```

### Test

トップレベルで
```
$ make test^rust
```
